# الأساسيات في Go

## المحتويات

| الملف | الموضوع |
|-------|---------|
| `hello_world.go` | مرحبا بالعالم |
| `variables.go` | المتغيرات والثوابت |
| `data_types.go` | أنواع البيانات |

## تشغيل الأمثلة

```bash
go run hello_world.go
go run variables.go
go run data_types.go
```

## المفاهيم المغطاة

- طباعة النصوص باستخدام `fmt`
- تعريف المتغيرات بطريقتين (`var` و `:=`)
- الثوابت (`const`)
- أنواع البيانات: int, float, string, bool
- المصفوفات والشرائح والخرائط
