# الدوال والحزم في Go

## المحتويات

| الملف | الموضوع |
|-------|---------|
| `functions.go` | الدوال والتوابع |
| `packages.go` | الحزم المدمجة |

## تشغيل الأمثلة

```bash
go run functions.go
go run packages.go
```

## المفاهيم المغطاة

- تعريف الدوال
- المعاملات والقيم المُعادة
- قيم عودة متعددة
- قيم عودة مسماة
- دوال variadic
- الدوال كقيم (First-class)
- الحزم المدمجة: fmt, math, strings, sort, time
