# 🔌 REST API - واجهة برمجية

واجهة برمجية REST بسيطة لإدارة المهام مبنية بلغة Go مع تخزين البيانات في الذاكرة.

## ✨ المميزات

- ✅ نقاط نهاية RESTful كاملة
- ✅ دعم عمليات GET و POST
- ✅ استجابات JSON
- ✅ نظام قفل للتزامن (thread-safe)
- ✅ بيانات أولية مُهيأة

## 📋 متطلبات التشغيل

- Go 1.21 أو أحدث

## 🚀 كيفية التشغيل

```bash
go run main.go
```

## 💡 نقاط النهاية المتاحة

| المسار | الطريقة | الوصف |
|--------|---------|-------|
| `/tasks` | GET | جلب جميع المهام |
| `/tasks` | POST | إنشاء مهمة جديدة |
| `/tasks/{id}` | GET | جلب مهمة محددة |

## 💻 أمثلة الاستخدام

```bash
# جلب جميع المهام
curl http://localhost:8081/tasks

# إنشاء مهمة جديدة
curl -X POST http://localhost:8081/tasks \
  -H "Content-Type: application/json" \
  -d '{"title": "مهمة جديدة"}'

# جلب مهمة محددة
curl http://localhost:8081/tasks/1
```

### مثال الاستجابة:

```json
[
  {
    "id": 1,
    "title": "تعلم Go",
    "completed": false,
    "created_at": "2024-01-15"
  }
]
```
