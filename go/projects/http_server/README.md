# 🌐 خادم HTTP - HTTP Server

خادم ويب بسيط مكتوب بلغة Go يوفر نقاط نهاية متعددة مع دعم JSON.

## ✨ المميزات

- ✅ خادم HTTP سريع وخفيف
- ✅ معالجة طلبات GET
- ✅ إرجاع استجابات JSON
- ✅ نقطة نهاية للتحقق من صحة الخادم
- ✅ دعم الكاراكترات العربية

## 📋 متطلبات التشغيل

- Go 1.21 أو أحدث

## 🚀 كيفية التشغيل

```bash
go run main.go
```

أو بناء وتشغيل:

```bash
go build -o http_server main.go
./http_server
```

## 💡 نقاط النهاية المتاحة

| المسار | الطريقة | الوصف |
|--------|---------|-------|
| `/` | GET | الصفحة الرئيسية |
| `/info` | GET | معلومات الخادم بصيغة JSON |
| `/health` | GET | فحص صحة الخادم |

## 💻 أمثلة الاستخدام

```bash
# الصفحة الرئيسية
curl http://localhost:8080/

# معلومات الخادم
curl http://localhost:8080/info

# فحص الصحة
curl http://localhost:8080/health
```

### مثال الاستجابة من `/info`:

```json
{
  "message": "مرحبا من Go HTTP Server",
  "time": "2024-01-15 10:30:00",
  "version": "1.0.0"
}
```
