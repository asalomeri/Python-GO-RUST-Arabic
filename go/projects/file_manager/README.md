# 📁 مدير الملفات - File Manager

أداة إدارة الملفات مكتوبة بلغة Go تتيح قراءة وكتابة والبحث في الملفات.

## ✨ المميزات

- ✅ قراءة محتوى الملفات
- ✅ كتابة محتوى للملفات
- ✅ عرض محتويات المجلدات
- ✅ البحث عن الملفات حسب الامتداد
- ✅ معالجة الأخطاء

## 📋 متطلبات التشغيل

- Go 1.21 أو أحدث

## 🚀 كيفية التشغيل

```bash
go run main.go
```

## 💡 أمثلة الاستخدام

```
=== مدير الملفات ===

✅ تم إنشاء الملف بنجاح

محتوى الملف:
مرحبا! هذا ملف تجريبي من Go
السطر الثاني
السطر الثالث

محتوى /tmp:
  📄 test_file.txt
  📁 systemd-private-...
```

## 📁 هيكل الملفات

```
file_manager/
├── main.go   # الكود الرئيسي
└── README.md # هذا الملف
```

## 🔧 الدوال المتاحة

| الدالة | الوصف |
|--------|-------|
| `قراءة_ملف(مسار)` | قراءة محتوى ملف نصي |
| `كتابة_ملف(مسار, محتوى)` | كتابة نص في ملف |
| `عرض_المجلد(مسار)` | عرض محتويات مجلد |
| `بحث_عن_ملفات(مجلد, امتداد)` | البحث عن ملفات بامتداد معين |

## 🔮 التحسينات المستقبلية

- إضافة نسخ ونقل الملفات
- إضافة ضغط وفك ضغط الملفات
- إضافة واجهة تفاعلية
- إضافة مراقبة تغييرات الملفات
