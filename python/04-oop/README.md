# البرمجة كائنية التوجه في بايثون

## المحتويات

| الملف | الموضوع |
|-------|---------|
| `classes.py` | الكلاسات والكائنات |
| `inheritance.py` | الوراثة وتعدد الوراثة |

## تشغيل الأمثلة

```bash
python classes.py
python inheritance.py
```

## المفاهيم المغطاة

- تعريف الكلاسات
- `__init__` و `__str__` و `__repr__`
- متغيرات الكلاس والكائن
- الخصائص (Properties)
- الوراثة البسيطة
- تعدد الوراثة
- `isinstance` و `issubclass`
