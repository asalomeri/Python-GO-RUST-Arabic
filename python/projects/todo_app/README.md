# ✅ تطبيق قائمة المهام - Todo App

تطبيق لإدارة المهام اليومية مكتوب بلغة Python مع حفظ البيانات في ملف JSON.

## ✨ المميزات

- ✅ إضافة مهام جديدة
- ✅ عرض جميع المهام مع حالتها
- ✅ تحديد المهام كمكتملة
- ✅ حذف المهام
- ✅ حفظ البيانات تلقائياً في ملف JSON
- ✅ استرجاع المهام عند إعادة التشغيل

## 📋 متطلبات التشغيل

- Python 3.6 أو أحدث
- لا توجد مكتبات خارجية مطلوبة (يستخدم `json` و `os` من المكتبة القياسية)

## 🚀 كيفية التشغيل

```bash
python todo.py
```

## 💡 أمثلة الاستخدام


## 📁 هيكل الملفات

```
todo_app/
├── todo.py       # الملف الرئيسي للتطبيق
└── tasks.json    # ملف حفظ المهام (يُنشأ تلقائياً)
```

## 🔧 الدوال المتاحة

| الدالة | الوصف |
|--------|-------|
| `تحميل_المهام()` | تحميل المهام من ملف JSON |
| `حفظ_المهام(مهام)` | حفظ المهام في ملف JSON |
| `إضافة_مهمة(مهام, عنوان)` | إضافة مهمة جديدة |
| `عرض_المهام(مهام)` | عرض جميع المهام |
| `إكمال_مهمة(مهام, رقم)` | تحديد مهمة كمكتملة |
| `حذف_مهمة(مهام, رقم)` | حذف مهمة |

- إضافة تواريخ الاستحقاق

- إضافة البحث والتصفية
