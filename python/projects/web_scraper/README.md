# 🌐 Web Scraper - أداة استخراج البيانات

أداة بسيطة لاستخراج البيانات من صفحات الويب باستخدام Python.

## ✨ المميزات

- ✅ جلب محتوى صفحات الويب
- ✅ استخراج الروابط من الصفحات
- ✅ فلترة الروابط حسب الكلمات المفتاحية
- ✅ إزالة الروابط المكررة
- ✅ معالجة الأخطاء
- ✅ دعم الكاراكترات العربية

## 📋 متطلبات التشغيل

- Python 3.6 أو أحدث
- المكتبات المطلوبة موجودة في `requirements.txt`

## 🚀 التثبيت والتشغيل

```bash
# تثبيت المتطلبات
pip install -r requirements.txt

# تشغيل البرنامج
python scraper.py
```

## 💡 أمثلة الاستخدام

```python
from scraper import جلب_صفحة, استخراج_الروابط, فلترة_الروابط

# جلب صفحة
محتوى = جلب_صفحة("https://example.com")

# استخراج الروابط
روابط = استخراج_الروابط(محتوى)

# فلترة الروابط
روابط_مفلترة = فلترة_الروابط(روابط, كلمة_مفتاحية="python")
print(روابط_مفلترة)
```

## 📁 هيكل الملفات

```
web_scraper/
├── scraper.py        # الملف الرئيسي
└── requirements.txt  # المكتبات المطلوبة
```

## 🔧 الدوال المتاحة

| الدالة | الوصف |
|--------|-------|
| `جلب_صفحة(رابط)` | جلب محتوى صفحة HTML |
| `استخراج_الروابط(محتوى_html)` | استخراج جميع الروابط |
| `فلترة_الروابط(روابط, كلمة)` | فلترة الروابط حسب كلمة مفتاحية |

## ⚠️ ملاحظات

- احترم شروط الاستخدام للمواقع التي تزورها
- لا تضغط على الخوادم بطلبات كثيرة ومتتالية
- بعض المواقع تمنع الـ scraping

- حفظ النتائج في ملف CSV أو JSON
- إضافة دعم الصفحات المتعددة
- إضافة proxy support
