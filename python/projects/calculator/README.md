# 🧮 الآلة الحاسبة - Calculator

آلة حاسبة تفاعلية بسيطة مكتوبة بلغة Python مع دعم كامل للعمليات الحسابية الأساسية.

## ✨ المميزات

- ✅ عملية الجمع
- ✅ عملية الطرح
- ✅ عملية الضرب
- ✅ عملية القسمة مع التحقق من القسمة على صفر
- ✅ واجهة تفاعلية بالعربية
- ✅ معالجة الأخطاء

## 📋 متطلبات التشغيل

- Python 3.6 أو أحدث
- لا توجد مكتبات خارجية مطلوبة

## 🚀 كيفية التشغيل

```bash
python calculator.py
```

## 💡 أمثلة الاستخدام

```
========================================
مرحبا بك في الآلة الحاسبة العربية
========================================

اختر العملية:
1. جمع
2. طرح
3. ضرب
4. قسمة
5. خروج

أدخل اختيارك (1-5): 1
أدخل الرقم الأول: 10
أدخل الرقم الثاني: 5
النتيجة: 15.0
```

## 📁 هيكل الملفات

```
calculator/
└── calculator.py   # الملف الرئيسي للآلة الحاسبة
```

## 🔧 الدوال المتاحة

| الدالة | الوصف | المثال |
|--------|-------|--------|
| `جمع(أ, ب)` | جمع عددين | `جمع(5, 3)` → `8` |
| `طرح(أ, ب)` | طرح عددين | `طرح(10, 4)` → `6` |
| `ضرب(أ, ب)` | ضرب عددين | `ضرب(6, 7)` → `42` |
| `قسمة(أ, ب)` | قسمة عددين | `قسمة(15, 3)` → `5.0` |

- إضافة سجل العمليات
- إضافة واجهة رسومية
