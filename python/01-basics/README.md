# الأساسيات في بايثون

## المحتويات

| الملف | الموضوع |
|-------|---------|
| `hello_world.py` | مرحبا بالعالم - أول برنامج |
| `variables.py` | المتغيرات وأنواعها |
| `data_types.py` | أنواع البيانات المختلفة |

## تشغيل الأمثلة

```bash
python hello_world.py
python variables.py
python data_types.py
```

## المفاهيم المغطاة

- طباعة النصوص
- تعريف المتغيرات
- أنواع البيانات: نصوص، أعداد، قوائم، قواميس، مجموعات، صفوف
- العمليات الحسابية الأساسية
