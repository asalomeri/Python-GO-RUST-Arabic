# الدوال في بايثون

## المحتويات

| الملف | الموضوع |
|-------|---------|
| `functions.py` | الدوال الأساسية والمتقدمة |
| `lambda.py` | دوال lambda المجهولة |

## تشغيل الأمثلة

```bash
python functions.py
python lambda.py
```

## المفاهيم المغطاة

- تعريف الدوال الأساسية
- المعاملات والقيم الافتراضية
- `*args` و `**kwargs`
- الدوال المتداخلة
- دوال lambda
- map, filter, reduce
