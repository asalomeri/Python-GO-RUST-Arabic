# التحكم في التدفق في بايثون

## المحتويات

| الملف | الموضوع |
|-------|---------|
| `if_else.py` | الشروط والتفريع |
| `loops.py` | الحلقات التكرارية |

## تشغيل الأمثلة

```bash
python if_else.py
python loops.py
```

## المفاهيم المغطاة

- جمل if/elif/else
- الشرط الثلاثي (Ternary)
- الشروط المركبة (and, or, not)
- match/case (Python 3.10+)
- حلقات for و while
- break و continue
- list comprehension
