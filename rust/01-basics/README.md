# الأساسيات في Rust

## المحتويات

| الملف | الموضوع |
|-------|---------|
| `hello_world.rs` | مرحبا بالعالم |
| `variables.rs` | المتغيرات والثوابت |
| `data_types.rs` | أنواع البيانات |

## تشغيل الأمثلة

```bash
rustc hello_world.rs && ./hello_world
rustc variables.rs && ./variables
rustc data_types.rs && ./data_types
```

## المفاهيم المغطاة

- `println!` للطباعة
- المتغيرات بـ `let`
- القابلية للتغيير بـ `mut`
- الثوابت بـ `const`
- Shadowing
- أنواع البيانات: الأعداد، النصوص، الصفوف، المصفوفات
