# الملكية والاستعارة في Rust

## المحتويات

| الملف | الموضوع |
|-------|---------|
| `ownership.rs` | نظام الملكية |
| `borrowing.rs` | الاستعارة والمراجع |

## تشغيل الأمثلة

```bash
rustc ownership.rs && ./ownership
rustc borrowing.rs && ./borrowing
```

## المفاهيم المغطاة

- قواعد الملكية في Rust
- النقل (Move) والنسخ (Copy/Clone)
- الاستعارة غير القابلة للتغيير (`&T`)
- الاستعارة القابلة للتغيير (`&mut T`)
- قواعد المقرض (Borrow checker)
- شرائح النص والمصفوفات
