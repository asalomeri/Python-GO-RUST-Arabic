# الهياكل والسمات في Rust

## المحتويات

| الملف | الموضوع |
|-------|---------|
| `structs.rs` | الهياكل والتعدادات |
| `traits.rs` | السمات والعمومية |

## تشغيل الأمثلة

```bash
rustc structs.rs && ./structs
rustc traits.rs && ./traits
```

## المفاهيم المغطاة

- تعريف `struct`
- تطبيق `impl` للدوال
- التعدادات `enum`
- `match` مع الأنماط
- تعريف `trait`
- تطبيق traits
- الأنواع العامة (Generics)
- trait objects مع `dyn`
