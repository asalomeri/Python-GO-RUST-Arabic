# 🎮 لعبة حجر ورقة مقص - Rock Paper Scissors

لعبة حجر ورقة مقص تفاعلية مكتوبة بلغة Rust للعب ضد الحاسوب.

## ✨ المميزات

- ✅ واجهة تفاعلية بالعربية
- ✅ اختيار عشوائي للحاسوب
- ✅ تتبع النتائج (انتصارات، خسائر، تعادلات)
- ✅ إمكانية اللعب عدة جولات
- ✅ عرض النتيجة النهائية

## 📋 متطلبات التشغيل

- Rust 1.60 أو أحدث
- Cargo

## 🚀 كيفية التشغيل

```bash
# بناء المشروع
cargo build

# تشغيل اللعبة
cargo run
```

## 💡 كيفية اللعب

```

الحاسوب اختار: مقص

النتيجة: ✅1 ❌0 🤝0
```

## 🎯 قواعد اللعبة

- 🪨 الحجر يكسر ✂️ المقص
- 📄 الورقة تغطي 🪨 الحجر
- ✂️ المقص يقطع 📄 الورقة
