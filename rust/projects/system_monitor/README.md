# 📊 مراقب النظام - System Monitor

أداة لمراقبة موارد النظام مكتوبة بلغة Rust تعرض معلومات الذاكرة ووقت التشغيل.

## ✨ المميزات

- ✅ عرض معلومات الذاكرة (RAM)
- ✅ عرض وقت تشغيل النظام (uptime)
- ✅ عرض الوقت الحالي
- ✅ معالجة الأخطاء بأمان
- ✅ تحويل الوحدات تلقائياً (KB إلى MB)

## 📋 متطلبات التشغيل

- Rust 1.60 أو أحدث
- Linux (للوصول إلى `/proc/meminfo` و `/proc/uptime`)

## 🚀 كيفية التشغيل

```bash
# بناء المشروع
cargo build

# تشغيل البرنامج
cargo run
```
