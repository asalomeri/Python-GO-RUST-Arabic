# 🛠️ أداة سطر الأوامر - CLI Tool

أداة سطر أوامر بسيطة مكتوبة بلغة Rust تدعم عمليات متعددة بواجهة عربية.

## ✨ المميزات

- ✅ أمر الترحيب بالاسم
- ✅ عملية الجمع بين عددين
- ✅ عكس النصوص
- ✅ عرض المساعدة
- ✅ معالجة الأخطاء

## 📋 متطلبات التشغيل

- Rust 1.60 أو أحدث
- Cargo

## 🚀 كيفية التشغيل

```bash
# بناء المشروع
cargo build

# تشغيل مع أمر
cargo run -- مرحبا أحمد
cargo run -- جمع 5 3
cargo run -- عكس "مرحبا"
cargo run -- مساعدة
```

## 💡 أمثلة الاستخدام

```
$ cargo run -- مرحبا أحمد
مرحبا يا أحمد! 👋

$ cargo run -- جمع 10 5
10 + 5 = 15

$ cargo run -- عكس "hello"
النص الأصلي: hello
النص المعكوس: olleh

$ cargo run -- مساعدة
أداة سطر الأوامر - CLI Tool

الاستخدام:
  cli_tool <أمر> [خيارات]

الأوامر:
  مرحبا <اسم>    - طباعة ترحيب
  جمع <أ> <ب>    - جمع عددين
  عكس <نص>       - عكس النص
  مساعدة         - عرض هذه المساعدة
```
